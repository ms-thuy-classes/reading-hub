# 📚 Learn with Ms. Thúy

**Practice Reading Smarter Every Day**

Website luyện đọc tiếng Anh chuyên nghiệp với PDF, câu hỏi tương tác và chấm điểm tự động.

## ✨ Tính năng
 
- 🎨 Giao diện Pastel Gradient + Glassmorphism hiện đại
- 🌙 Dark Mode
- 📱 Responsive hoàn hảo (Mobile/Tablet/Desktop)
- 📄 PDF Viewer chất lượng cao với PDF.js
- 🖍️ Highlight PDF
- ❓ 5 loại câu hỏi: MCQ, Fill, Matching, Heading, Checkbox
- 📊 Chấm điểm tự động thang 10
- 💾 Auto-save đáp án
- ⏱️ Đếm thời gian làm bài
- 🔖 Bookmark & Favorite
- 📜 Lịch sử điểm
- 🖨️ In & Export kết quả
- 📴 Offline support (PWA)
